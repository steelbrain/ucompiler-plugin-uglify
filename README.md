UCompiler-Plugin-Uglify
======================

This plugin for uglify makes sure that your generated files always end with a new line.
It also normalizes their EOLs to LF.


#### How to Use

Simply add "uglify" to `plugins` field in your `.ucompilerrc`

```js
{
  plugins: ["uglify"]
}
```
